# Event Management System

The Event Management System is a full-stack web application that allows users to explore and book events, while Organizers and Admin can manage events, users, and platform activities through dedicated dashboards.

This project is developed as a real-world application and has been successfully deployed online.

----------------

## Live Project

- Live Website: https://uttsavevent.free.nf/
- GitHub Repository: https://github.com/Jaydip2999/uttsav-event-management

--------------

## Project Overview

The system provides role-based access for three different types of users.

### User
- User registration and login  
- Browse available events  
- View event details  
- Book events  
- View booking history  

### Organizer
- Organizer dashboard  
- Create and manage events  
- View bookings  
- Track earnings  

### Admin
- Admin dashboard  
- Manage users and organizers  
- Monitor all events  
- View bookings and platform activity  
- Revenue and commission tracking  

-----------------

## Key Features

- Role-based authentication (User / Organizer / Admin)  
- Responsive and user-friendly interface  
- Secure database integration using PHP and MySQL  
- Live deployed application
